Every job follows the same contract

class MyETLJob(ETLInterface):

    def extract(self):
        # read from source

    def transform(self):
        # clean, join, aggregate

    def load(self):
        # write output

    def run(self):
        with mlflow.start_run():
            self.extract()
            self.transform()
            self.load()

Solutions & Takeaways What the completed pipeline looks like

The four TODOs — solved

T1
Join chain .join(dim_date, …, "inner") then filter by date, then .join(dim_customer, …), then four "left" joins for the facts
T2
COALESCE amounts F.coalesce(F.col("AMT_Travel"), F.lit(0)) — applied right after the joins, before any aggregation
T3
Aggregate .groupBy(date, agency, country, class, customer_type)
.agg(F.countDistinct(…), F.round(F.sum(…), 2), …)
T4
Revenue tier F.when(col >= 2000, "High").when(col >= 500, "Medium").otherwise("Low")
